// Beispiel:
//
// **Lookahead:** Matcht, wenn nach dem aktuellen Ausdruck ... folgt.
// Beispiel: `abc(?=def)`
// [Info: Kann Backtracking verursachen]
//
// **Digit:** Matcht eine einzelne Ziffer (0-9).
// Beispiel: `\d+` matcht '123' in 'abc123def'.
// [Info: Sehr schnell, keine Nebenwirkungen]
